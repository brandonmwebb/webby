Review each task file in the tasks/ folder one by one. For each task:

1. Read the task instructions
2. Analyze if there's a better way to accomplish the task
3. If you find an optimization, modify the task file to improve it
4. Document any changes made in tasks/learnings.log with:
   - Task name
   - What was changed
   - Why it's better
   - Timestamp

Only make changes if there's a clear improvement. Don't change things just for the sake of changing them.
